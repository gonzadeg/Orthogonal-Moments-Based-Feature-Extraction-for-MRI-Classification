# Orthogonal-Moments-Based-Feature-Extraction-for-MRI-Classification
Brain tumor classification from MRI images using orthogonal moments for feature extraction and a Random Forest model for multiclass classification. Achieves ~96.5% accuracy across glioma, meningioma, pituitary, and non-tumor cases.

## 📄 Publication
This project is based on a published research paper:

**Feature extraction based on orthogonal moments for brain tumor classification in MRI**
